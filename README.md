# AGEROUTE360 - Digital Road Infrastructure Platform

A modern, responsive web platform for national road infrastructure management, featuring project tracking, interactive maps, tender opportunities, and transparency reports.

## 🚀 Tech Stack

- **React 18** - Modern UI library
- **Vite** - Lightning-fast build tool
- **React Router v6** - Client-side routing
- **Tailwind CSS v3** - Utility-first styling
- **GSAP + ScrollTrigger** - Advanced animations
- **Framer Motion** - Page transitions and interactions
- **Mapbox GL JS** - Interactive maps
- **React Query** - Data fetching and caching
- **React Helmet Async** - SEO management

## 📦 Installation

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
